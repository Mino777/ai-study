# 🏔️ 이직 등반전 (The Ascent) — 게임 시스템

> **컨셉:** 3개월 안에 3개의 탑을 오르고 최종 보스(실전 면접)를 잡는 RPG.
> 주인공은 **실무 스탯 만렙 전사 / 시험 마법 Lv.1** 언밸런스 캐릭.
> **목표:** 매일 조금씩, 재밌게, 안 질리게. 실력은 결과일 뿐 — 진짜 게임은 *꾸준함*이다.

> **Claude 운영 규약:** 매 세션 종료 시 이 파일의 캐릭터 시트를 갱신(XP·레벨·스탯·업적·스트릭). 세션 시작 시 캐릭터 시트를 보여주며 "오늘의 퀘스트"를 던진다. 보상은 후하게, 하지만 정직하게 (안 한 건 XP 없음).

## 🔥 스트릭 & 콤보

연속 학습일이 쌓이면 **콤보 배율**이 붙어 그날 XP가 증폭된다. 하루 빠지면 콤보 리셋(스트릭은 유지되되 배율만 초기화).

| 연속일 | 콤보 배율 |
|--------|-----------|
| 1–2일 | x1.0 |
| 3–6일 | x1.2 |
| 7–13일 | x1.5 |
| 14–29일 | x1.8 |
| 30일+ | x2.0 🔥 |

## 🐉 보스전 (모의 면접)

각 탑의 층을 다 오르면 보스가 기다린다. 보스전 = 실전 모의면접.

| 보스 | 위치 | 시험 영역 | 상태 |
|------|------|-----------|------|
| 👾 옵셔널 크래셔 | 제1탑 정상 | Swift/동시성 기초 기술면접 | 🔒 미개방 |
| 🐙 동시성 히드라 | 제2탑 정상 | async/await·아키텍처 심화 | 🔒 미개방 |
| 🗿 시스템 디자인 골렘 | 제3탑 중턱 | 모바일 시스템 디자인 | 🔒 미개방 |
| 👹 최종 면접관 | 제3탑 정상 | 종합 실전 면접 (기술+행동) | 🔒 미개방 |

## 🎲 특별 규칙 (재미 장치)

- **🗺️ 사이드 퀘스트 (백엔드 대륙):** 매 세션 후반 5~7분, `BACKEND.md` 지도 노드 1개 개방 (+8 XP). iOS 가로축 넓히기. 상세는 `BACKEND.md`.
- **주간 랜덤 이벤트:** 매주 월요일, 제가 랜덤 버프/도전과제 하나 던짐 (예: "이번 주 알고리즘 3문제 클리어 시 +30 보너스")
- **리롤 토큰:** 컨디션 안 좋은 날은 "오늘은 15분 미니 퀘스트만" 선언 가능 (스트릭 유지, 콤보만 x1.0)
- **자율 부활:** 하루 빠져도 다음 날 이어가면 OK. 죄책감 게임화 금지 — 완주가 승리 조건.
