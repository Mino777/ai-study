# 🗺️ 백엔드 대륙 탐험 (사이드 퀘스트)

> **목적:** iOS 엔지니어의 가로축 — 백엔드/DB 큰그림. "화면만 아는 사람" → "시스템을 아는 사람".
> **방식:** 매 세션 딱 **1노드(~7분, 그림 위주)**. 모든 개념을 iOS가 이미 만지는 것(URLSession·이미지 로딩·무한스크롤·푸시)에 앵커링.
> **보상:** 노드 1개 = +8 XP. 대륙 1개 완주 = 업적 + 보너스.

> **Claude 운영 규약:** 세션 후반 5~7분에 "오늘의 백엔드 한 조각"으로 다음 미개방 노드 1개를 그림으로 설명. 설명 후 이 파일에서 해당 노드 `[ ]`→`[x]`, 아래 "현재 위치" 갱신, GAME.md 🌐 백엔드 스탯 +1~2.
