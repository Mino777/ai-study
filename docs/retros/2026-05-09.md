- **Blake 스카우트 자동 머지가 처음부터 없었다**: daily-scout-blakecrosley.yml 만들 때 auto-lesson-geeknews.yml 의 자동 머지 패턴을 확인하지 않았다 (다른 세션의 판단 실수이지만, 이번 세션에서도 PR 쌓인 걸 늦게 발견).
- **인터뷰 디벨롭을 이 세션에서 시작하지 못했다**: 컨텍스트 60% 임계값에 도달하여 다음 세션으로 이관.
