## 아쉬운 것

- **Gemini 매칭 품질을 로컬에서 검증 못 함**: API 키가 GitHub Secrets에만 있어서 RSS 파싱만 확인하고 Gemini 매칭 결과는 CI에서 첫 실행 때 확인해야 함. dry-run에 mock 응답을 넣거나 로컬 .env를 만들었으면 더 확실했을 것
