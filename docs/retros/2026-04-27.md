- **근본 원인 추적이 정확했다**: 빌드 에러 → JSX 중괄호 → 중첩 코드 펜스 파싱 오류까지 2단계 원인 추적
- **3중 방어 설계**: 프롬프트 가드만 의존하던 구조에서 자동수정 + 빌드차단 + GHA게이트로 체계화

## 아쉬운 것

- **첫 배포에서 파일 존재 확인을 안 함**: `git pull` 전에 로컬에서 파일이 없다고 판단하고 빌드 통과로 착각. 리모트 상태와 로컬 상태가 다른 걸 확인하지 않은 내 판단 실수
- **중첩 코드 펜스 케이스를 첫 수정에서 놓침**: `escapeJsxCurlyBraces` 작성 시 단순 토글로 구현 → 배포 후 재실패. 코드 블록 파싱 로직을 작성할 때 중첩 가능성을 고려했어야 함

## 다음에 적용할 것
