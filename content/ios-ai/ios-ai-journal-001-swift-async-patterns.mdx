## iOS AI Journal 001: Swift async/await로 컨텍스트 수집하기

Journal 000의 정적 구조를 벗어나, **실제로 여러 소스에서 동시에 컨텍스트를 수집**하는 패턴을 다룬다.

### 문제: 컨텍스트 수집이 느리다

여러 소스의 데이터를 순차적으로 수집하면:
```
UserDefaults (50ms) 
  ↓
CoreLocation (200ms)  
  ↓
Contacts API (300ms)
  ↓
Total: 550ms 이상
```

### 해결: async/await로 병렬 수집

```swift
struct ContextCollector {
    async func collectContext() -> AIContext {
        async let sessionData = fetchSessionData()
        async let locationData = fetchLocation()
        async let recentContacts = fetchRecentContacts()
        
        let (session, location, contacts) = await (
            sessionData, locationData, recentContacts
        )
        
        return AIContext(
            sessionId: session.id,
            userLocation: location,
            recentInteractions: contacts,
            timestamp: Date()
        )
    }
}
```

### 성능: 병렬 수집 vs 순차 수집

**순차 수집**: 550ms
**병렬 수집**: 300ms (가장 느린 작업 기준)

**54% 시간 단축** = 사용자가 더 빨리 AI 응답을 받음

### 주의점

1. **타이밍 에러**: 한 소스가 실패하면 전체 컨텍스트 수집 실패 → try/catch 필수
2. **메모리 누수**: 수집 중 앱이 포그라운드 → 백그라운드로 전환되면 문제 발생
3. **권한 체크**: Location, Contacts는 사용자 권한이 필요 → 미리 확인
