## 한 줄 요약

Apple Intelligence는 `IntelligenceKit` 같은 전용 프레임워크가 없다. 기존 **App Intents**, **CoreSpotlight**, **UITextView Writing Tools**를 통해 Apple Intelligence와 앱이 연동된다.

---

## Apple Intelligence 연동의 3가지 축

```
앱 개발자가 Apple Intelligence와 연동하는 방법

1. App Intents      → Siri / Action Button / Spotlight에 앱 기능 노출
2. CoreSpotlight    → 앱 내 콘텐츠를 Apple Intelligence가 검색 가능하게
3. Writing Tools    → UITextView에 자동으로 제공 (별도 구현 불필요)
```

---

## 1. App Intents — Siri에 앱 기능 노출

iOS 16+에서 도입된 **App Intents** 프레임워크. iOS 18에서 Apple Intelligence와 통합되어 Siri가 앱의 기능을 자연어로 실행할 수 있게 된다.

### 기본 AppIntent 구현

```swift
import AppIntents

// Siri가 실행할 수 있는 Intent 정의
struct OpenCategoryIntent: AppIntent {
    static var title: LocalizedStringResource = "카테고리 열기"
    static var description = IntentDescription("특정 카테고리의 항목 목록을 열기")

    @Parameter(title: "카테고리")
    var category: String

    func perform() async throws -> some IntentResult {
        // 앱의 실제 화면 열기 로직
        await MainActor.run {
            NotificationCenter.default.post(
                name: .openCategory,
                object: nil,
                userInfo: ["category": category]
            )
        }
        return .result()
    }
}
```

### iOS 18: @AssistantSchemas

iOS 18에서 추가된 `@AssistantSchemas`를 사용하면 Apple Intelligence가 앱의 Intent를 더 잘 이해한다.

```swift
import AppIntents

// iOS 18+: AssistantSchemas로 표준 스키마 채택
struct CreateNoteIntent: AppIntent {
    static var title: LocalizedStringResource = "메모 만들기"

    @Parameter(title: "제목")
    var title: String

    @Parameter(title: "내용")
    var content: String?

    func perform() async throws -> some IntentResult & ProvidesDialog {
        let note = try await NoteRepository.create(title: title, content: content)
        return .result(dialog: "'\(title)' 메모를 만들었습니다.")
    }
}

// AppShortcutsProvider: 앱의 주요 단축어 등록
struct MyAppShortcuts: AppShortcutsProvider {
    static var appShortcuts: [AppShortcut] {
        AppShortcut(
            intent: CreateNoteIntent(),
            phrases: [
                "\\(.applicationName)에 메모 만들기",
                "\\(.applicationName)에서 \\(.title) 메모 추가"
            ],
            shortTitle: "메모 만들기",
            systemImageName: "note.text"
        )
    }
}
```

## 2. CoreSpotlight — 콘텐츠를 Apple Intelligence가 검색하게

앱 내 콘텐츠를 Spotlight에 인덱싱하면 Apple Intelligence가 "지난주에 저장한 요리 레시피"처럼 의미 기반으로 찾을 수 있다.

```swift
import CoreSpotlight

// 앱 콘텐츠 Spotlight 인덱싱
func indexItems(_ items: [MyItem]) {
    let searchableItems = items.map { item -> CSSearchableItem in
        let attributes = CSSearchableItemAttributeSet(contentType: .text)
        attributes.title = item.title
        attributes.contentDescription = item.description
        attributes.keywords = item.tags
        attributes.thumbnailData = item.thumbnailData

        return CSSearchableItem(
            uniqueIdentifier: item.id.uuidString,
            domainIdentifier: "com.myapp.items",
            attributeSet: attributes
        )
    }

    CSSearchableIndex.default().indexSearchableItems(searchableItems) { error in
        if let error {
            print("Spotlight 인덱싱 실패: \\(error)")
        }
    }
}

// 항목 삭제 시 인덱스에서도 제거
func removeFromIndex(itemId: String) {
    CSSearchableIndex.default().deleteSearchableItems(
        withIdentifiers: [itemId]
    ) { error in
        if let error {
            print("Spotlight 인덱스 삭제 실패: \\(error)")
        }
    }
}
```

### NSUserActivity로 화면 인덱싱

```swift
// 특정 화면을 Siri / Spotlight에서 직접 열 수 있게
class DetailViewController: UIViewController {
    var item: MyItem?

    override func viewDidAppear(_ animated: Bool) {
        super.viewDidAppear(animated)
        guard let item else { return }

        let activity = NSUserActivity(activityType: "com.myapp.viewItem")
        activity.title = item.title
        activity.userInfo = ["itemId": item.id.uuidString]
        activity.isEligibleForSearch = true
        activity.isEligibleForPrediction = true  // Siri 예측에 포함

        let attributes = CSSearchableItemAttributeSet(contentType: .text)
        attributes.contentDescription = item.description
        activity.contentAttributeSet = attributes

        self.userActivity = activity
        activity.becomeCurrent()
    }
}
```

## 3. Writing Tools — UITextView 자동 통합

iOS 18에서 **UITextView**에 Writing Tools(교정, 요약, 다시 쓰기 등)가 자동으로 제공된다. **별도 구현이 필요 없다.**

```swift
// UITextView 기본 설정만으로 Writing Tools 활성화됨
let textView = UITextView()
textView.isEditable = true
// iOS 18에서 자동으로 Writing Tools 메뉴 제공

// 특정 시나리오에서 비활성화하고 싶다면
if #available(iOS 18, *) {
    textView.writingToolsBehavior = .none  // 비활성화
    // 또는 .limited (일부 기능만)
    // 또는 .complete (기본값, 전체 활성화)
}
```

## 주의사항

**존재하지 않는 API**:
- `IntelligenceKit` — 실제 존재하지 않는 가상 프레임워크
- `IntelligenceContextRepresentable` — 없음
- `IntelligenceActionType` — 없음

Apple Intelligence 연동은 기존 프레임워크(App Intents, CoreSpotlight, NSUserActivity)를 통해 이루어진다. Apple이 앱의 데이터를 직접 "가져가는" API는 공개되어 있지 않고, 앱이 적절한 포맷으로 시스템에 등록/기여하는 방식이다.
